NewGrain is a Telegram bot (discovery / Г1 phase) where an agronomist uploads field photos that a backend stores for an ML labeling/training pipeline. Built per `tech_spec_v3.docx` (in iCloud SFAI folder).

**Single success metric:** the agronomist uploads 15–30 photos/week for 12 weeks without reminders.

**Guiding principle:** build to learn, not to scale (spec §3.3) — no analytics, no gamification, single farm, no web cabinet. Reject scope that doesn't serve the learning goal. See [[newgrain-status-2026-05]] for current build state.
